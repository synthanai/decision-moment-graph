# நூல் / NOOL — DMG Open Standard (C3)

## நோக்கம் / Intent
1. No open standard for decision provenance exists
2. AI agents produce decisions without auditable format
3. Regulatory pressure (EU AI Act) demands decision traceability
4. DMG's 8 primitives provide a minimal viable standard

**Purpose:** Design science paper presenting DMG as an open standard for decision provenance, with schema specification, worked examples, and interoperability analysis.

## வடிவம் / Abstraction
**DESIGN SCIENCE**: Present DMG as a standards artifact (Hevner 2004) evaluated against completeness, interoperability, and adoptability criteria.

### The 8 Primitives
| Primitive | Function |
|-----------|----------|
| MEMO | The decision brief |
| MOMENT | The event stream |
| RAMP | Risk calibration level |
| DOORS | Reversibility gates |
| DISSENT | Structured disagreement records |
| TRACE | Evidence links |
| COMMIT | State transitions |
| OUTCOME | Learning and results |
