# நூல் / NOOL — Decision Moment Graph (DMG)

> *The Reasoning Thread: a record of INTENT (why), ABSTRACTION (what type), and CHAIN (how).*

---

## நோக்கம் / Intent (Soul: WHY)

Store decision moments, not raw chats. DMG is the reference implementation of the Decision Moment Standard (DMS), providing schemas, SDKs, and validators to ensure decisions are MERIT-compliant (Measured, Evidenced, Reversible, Inspectable, Traceable). "DMS defines the rules. DMG stores the proof."

## வடிவம் / Abstraction (Mind: WHAT TYPE)

**DESIGN**: An open standard graph encoding implementing the 7-Step Heptagon lifecycle (FRAME → SPAR → GATE → COMMIT → ENACT → YIELD → GAUGE). Governance gates scale with RAMP level. MCP Context Engine provides `validate_moment` and `create_moment_scaffold` tools. GAUGE Emitter and Decision Replay modules enable counterfactual analysis.

## சங்கிலி / Chain (Body: HOW)

`pip install decision-moment-graph` → `lifecycle.py` CLI for full lifecycle runs → `AgenticSPARAdapter` for programmatic integration. JSON Schema definitions in `dmg/schema/`, conformance test fixtures in `conformance/`. The kit is free, ARANGAM provides premium hosted infrastructure.
