# M0: Start Here

Welcome to the **DMG (Decision Moment Graph)** course.

By the end of this course, you will:
- Create MEMOs that teams actually use
- Build MOMENT provenance that stands up to scrutiny
- Apply proportional governance (RAMP/DOORS)
- Capture productive dissent
- Close the learning loop with OUTCOME checks

---

## What is DMG?

> **Stop storing chats. Store decision objects.**

DMG is an open standard for decision governance. It makes decisions:
- **Structured** — not narratives
- **Auditable** — provenance chain
- **Reversible** — governance gates
- **Learnable** — outcome loops

## The Mindset Shift

| From | To |

## Prerequisites

- A real decision you need to make (or recently made)
- 15-30 minutes per module
- Willingness to write things down

---

## How to Learn (Do → See → Name)

Each module follows the same pattern:

1. **Do**: Create an artifact (MEMO, MOMENT event, etc.)
2. **See**: Compare weak vs strong examples
3. **Name**: Understand the concept behind what you built

Don't memorize definitions. Build artifacts.

---

## Your First MEMO (5 minutes)

Before Module 1, try this:

Think of a decision you're facing. Write down:

1. **What's the decision?** (one sentence)
2. **What are 3 options?** (including "do nothing")
3. **Who owns this decision?**
4. **When should we review it?**

That's a MEMO skeleton. You'll build on this throughout the course.
