# Attribution Requirements

DMG (Decision Moment Graph) is open source under the MIT License. You are free to use, modify, and distribute this software.

## Attribution Guidelines

If you use DMG in your project, product, or service, we kindly request (but do not require) the following:

### For Software Projects

Include one of the following in your README, documentation, or about page:

**Badge (Preferred):**
```markdown
[![DMG Standard](https://img.shields.io/badge/DMG-Standard-purple)](https://github.com/synthanai/dmg-open-standard)
```

**Text:**
```
Built on DMG Standard — https://github.com/synthanai/dmg-open-standard
```

### For Academic Work

Please cite:
```
DMG: Decision Moment Graph — Open Standard for Decision Governance
SYNTHAI TECH PTY LTD, 2026
https://github.com/synthanai/dmg-open-standard
```

### For Commercial Products

If you build a commercial product using DMG:

1. **Attribution**: Include "Built on DMG Standard" in your product's documentation
2. **Certification**: For official DMG certification, see [DMG Certification Program](https://arangam.com/certification)
3. **Consider Premium**: For hosted MOMENT ledger, team collaboration, and enterprise governance, see [ARANGAM Platform](https://arangam.com)
