# DMG MCP Apps

Interactive UI for the Decision Moment Graph 7-step Heptagon lifecycle.

## UI Resources

| URI | Description |
|-----|-------------|
| `ui://dmg-kit/decision-flow` | 7-step lifecycle visualizer with MERIT compliance |

## Usage

```javascript
import { registerDmgAppsResources } from './mcp-apps/decision-flow.js';
registerDmgAppsResources(server);
```

## Preview

```bash
npx serve mcp-apps/
```
